Django Notice Board Application (with unit tests)

A simple Django application for creating, viewing, editing, and deleting notices. Built with the Django MVT pattern and uses SQLite as the default database.

FEATURES -Create notices -List all notices -View individual notices -Update notices -Delete notices

TECH STACK -Python 3 -Django Framework -SQLite3 (default Django DB) -HTML Templates with inheritance -Django Forms for input handling

ADMIN ACCESS Enable Django admin to manage authors and notices by: createsuperuser

ADMIN PANEL http://127.0.0.1:8000/admin/
